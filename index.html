Notifiles

Open source, local-first, git-based message templating made right. An open source library for writing platform agnostic email, text message and push notifications templates. Every template is consolidated, self contained and can be published to numerous online platforms including Minio and Bunny.net. Take control of your templates without concessions.

Sole source of truth

The JSON based form declaration is what separates Notifiles from other frameworks. Simply give. The markdown based local posts are standard.

Git based & Archival

Every input component follows a unique API that make it readable, testable, maintainable and shippable.

Multi-platforms

Host your templates on Minio, Bunny, and more (upcoming) and access them easily on your server.

Quick start üöÄ

Let's build a notifiles project.

1. Create a project

npx notifiles app new

2. Add a cloud storage (minio)

npx notifiles cloud add --cloudType minio

4. Create a template

npx notifiles template new

5. Build and sync the project to minio

npx notifiles sync
